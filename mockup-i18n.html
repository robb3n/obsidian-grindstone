Grindstone — 语言切换 Mockup / Language-Selectable Mockup

当前 UI 是分层双语：英文 ALL-CAPS 做装饰 eyebrow，中文做主内容。目标是改成"用户在设置里选一种语言（ZH / EN），UI 纯一种语言呈现"。 本 mockup 比对 7 个视图在两种模式下的最终样貌——所有装饰性英文 eyebrow（TODAY、DUE、PRE-FLIGHT 之类）一律丢弃，文字翻译对收敛为单一语言。 第八节展示新增的 Settings → 界面 → Language 控件位置。

7 · Onboarding modal

首次启动 modal 当前是 WELCOME · 欢迎 eyebrow + "磨石 Grindstone" 双语标题，section 头是 触发标签 / TRIGGER TAG，写入模式是 { zh, en, desc } 三件套。 单语化后：标题单语，section 头单行，模式选项仅留 zh-or-en 标题 + desc。RECOMMENDED 徽章保留单语。

注意：onboarding modal 在用户首次启动时呈现——彼时 Language 设置还未存在，所以默认语言需要通过 (a) 浏览器/OS 语言探测 (navigator.language) 或 (b) modal 顶部放一个语言切换器二选一。下方两版各示意一种系统语言导向。

ZH MODE

与笔记的交互方式

完全只读

嵌入卡片 ID推荐
在触发标签行尾添加 <!-- gs:xxxxxxxx --> 注释（一次性）。复习历史在重命名、移动、编辑后依然保留。

Note writeback
